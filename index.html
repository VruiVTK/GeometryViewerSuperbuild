Welcome to VruiVTKSuperbuild

This is the superbuild code that helps build VruiVTK along with all its pre-requisites including:

  • Vrui
  • VTK with the vtkRenderingExternal module turned ON
  • TCLAP

Clone VruiVTKSuperbuild source

VruiVTKSuperbuild can be built from source obtained using one of the download links on this page or using the commandline as follows:

$ git clone https://github.com/VruiVTK/VruiVTKSuperbuild.git

The above command creates a directory VruiVTKSuperbuild with the source code.

Configure Instructions

Configure the build using CMake keeping default options. Once configured, press 'Generate' to generate the makefile.

Build Instructions

Build using the make tool.

$ make

Run VruiVTK

Building the superbuild generates the application executable at <VruiVTKSuperbuild-build-dir>/VruiVTK/bld/ One can run the application by following the instructions provided here.
