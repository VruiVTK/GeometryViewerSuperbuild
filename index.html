Welcome to GeometryViewerSuperbuild

This is the superbuild code that helps build GeometryViewer along with all its pre-requisites including:

  • Vrui
  • VTK with the vtkRenderingExternal module turned ON

Clone GeometryViewerSuperbuild source

GeometryViewerSuperbuild can be built from source obtained using one of the download links on this page or using the commandline as follows:

$ git clone https://github.com/VruiVTK/GeometryViewerSuperbuild.git

The above command creates a directory GeometryViewerSuperbuild with the source code.

Configure Instructions

Configure the build using CMake keeping default options. Once configured, press 'Generate' to generate the makefile.

Build Instructions

Build using the make tool.

$ make

Run GeometryViewer

Building the superbuild generates the application executable at <GeometryViewerSuperbuild-build-dir>/GeometryViewer/bld/ One can run the application by following the instructions provided here.
